Calculator/Sequencer

Overview

A mobile app that combines a fully functional calculator with an interactive drum sequencer. At its core, it operates as a calculator but introduces playful musical elements. Each number button triggers a corresponding drum sound, turning basic calculations into an auditory experience. Additionally, the display's number string can be looped as a sequencer, allowing users to generate rhythmic patterns through calculations.

Designed in Figma · Built with Swift
